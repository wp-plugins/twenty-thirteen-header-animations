Spice up the header of your out-of-the-box Twenty Thirteen install! Instead of the default graphic images in the header, this plugin generates custom shapes on every page load, with an animation on hover. 

== Installation ==

Install, activate, enjoy.

== Frequently Asked Questions ==

= Hey this only replicates one of the three headers included in Twenty Thirteen. What gives? =

The others weren't as obvious as to animations that could be played with. If you're a developer and want to do something, go for it and show me.
